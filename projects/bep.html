Building Engaging Prototypes

Design Portfolio.

On my third module of classes, for the course Building Engaging Prototypes, we were given 6 challenges. These weekly challenges were all related to prototyping techniques. The final delivery was a design portoflio that included reports on all the six challenges.

Since the portfolio needed to be available online, and the choice on where to host it was given to us, I decided to code it myself. The project was valuable not only in learning more on how to apply better prototyping techniques and tools, including sound and graphic editing software, but I also learned quite a lot about CSS Grids and navigation usability while coding the portfolio.
